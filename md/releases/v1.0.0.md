The first **official** release of Rest Stop.

This version contains the _following_ features:

1.  A simple interface to create restic backups.
2.  Versioning of backups using restic.
3.  Support for different backends including several implemented through rclone.
4.  Support for exporting application settings and configuration.
5.  The ability to restore backups.
6.  The ability to restore backup configurations or application configuration.
7.  The ability to preview notifications.
8.  Resilient backups with the ability to re-initialize after failure.
9.  The ability to choose the frequency and number of backup versions created.
10.  The ability to operate silencing the background.
11.  Backup utility initializing on start up.
12.  Choose between light and dark interfaces.
13.  The ability to independently manage the triggering of different backups.
